# fivem-weed-macro

A simple macro for FiveM that automatically presses a selected key every 30 seconds.

---

## ⚙️ Requirements

You need AutoHotkey installed to run this script.

---

## 🚀 Features

- Automatically presses a key every 30 seconds
- Lightweight and easy to use
- Fully customizable

---

## 🎮 Controls

- **Start Key:** The key used to start the macro  
- **Stop Key:** The key used to stop the macro  
- **Exit Key:** Closes the script  

---

## 🔧 Customization

You can change the keys directly inside the script file.

```ahk
; >>> CHANGE THIS KEY <<<
R::  ; Start Key

; >>> CHANGE THIS KEY <<<
I::  ; Stop Key
